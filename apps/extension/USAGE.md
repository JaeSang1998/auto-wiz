# 사용 가이드 📖

## 기본 사용법

### 1. 확장 프로그램 설치
1. `npm run dev` 실행
2. Chrome에서 `chrome://extensions` 열기
3. **개발자 모드** 활성화
4. **압축해제된 확장 프로그램 로드** → `.output/chrome-mv3` 선택

### 2. 엘리먼트 선택 및 레코딩 (Alt + Shift)

#### 호버 모드
- 마우스를 움직이면 자동으로 엘리먼트가 **초록 테두리**로 하이라이트됩니다
- 툴바가 마우스 커서를 따라다닙니다 (희미하게 표시)
- 이 상태에서는 버튼을 클릭할 수 없습니다

#### 🔒 잠금 모드 (Alt + Shift)
1. 원하는 엘리먼트 위에서 **Alt + Shift (또는 Option + Shift)를 누릅니다**
2. 엘리먼트가 **주황 테두리 (3px)**로 변경됩니다
3. 툴바가 **활성화** (주황 테두리 + 강한 그림자)
4. 우측 상단에 "🔒 Locked" 표시
5. 이제 **버튼을 클릭**할 수 있습니다!
6. **액션 선택** 시 자동 해제 또는 **ESC/Alt+Shift**로 수동 해제

💡 **Tip**: Windows는 Alt + Shift, macOS는 Option + Shift로 누르세요!

### 3. 액션 버튼 (컬러풀 디자인)

#### 👆 Click (파랑)
- 선택한 엘리먼트를 클릭하는 액션 레코드
- 사용 예: 버튼, 링크 클릭

#### ⌨️ Type (초록)
- 선택한 input/textarea에 텍스트 입력
- 프롬프트가 나타나면 입력할 텍스트를 입력

#### 📄 Extract (주황)
- 선택한 엘리먼트의 텍스트를 추출
- innerText를 기본으로 추출

#### ⏱️ Wait (보라)
- 선택한 엘리먼트가 나타날 때까지 대기
- 타임아웃: 5초

### 4. 사이드패널 사용

#### 사이드패널 열기
1. Chrome 우측 상단 확장 프로그램 아이콘 클릭
2. 또는 브라우저 우측 사이드패널 영역 클릭

#### 버튼 설명
- **▶ Run**: 현재 탭에서 레코드된 플로우 실행
- **📤 Send**: 백엔드 API로 플로우 전송
- **⏸ Pause / ▶ Resume**: 픽커 일시 정지/재개
- **🗑 Reset**: 모든 레코드 초기화

### 5. 실전 예제

#### 예제 1: Google 검색 자동화
```
1. google.com 접속
2. 검색창 위에 마우스 이동
3. Alt + Shift 누름 🔒
4. ⌨️ Type 클릭 → "WXT framework" 입력
5. 검색 버튼 위로 마우스 이동
6. Alt + Shift 누름 🔒
7. 👆 Click 클릭
8. 사이드패널에서 ▶ Run 실행
```

#### 예제 2: 로그인 플로우
```
1. 로그인 페이지 접속
2. 이메일 입력 필드 → Alt+Shift → ⌨️ Type → 이메일 입력
3. 패스워드 입력 필드 → Alt+Shift → ⌨️ Type → 비밀번호 입력
4. 로그인 버튼 → Alt+Shift → 👆 Click
5. 환영 메시지 → Alt+Shift → 📄 Extract
```

## 단축키

| 키 | 동작 |
|---|---|
| **Alt + Shift** (Windows) | 툴바 잠금/해제 토글 |
| **Option + Shift** (macOS) | 툴바 잠금/해제 토글 |
| **ESC** | 잠금 해제 |
| **마우스 이동** | 엘리먼트 하이라이트 (잠금 상태가 아닐 때) |

## 팁

1. **Alt + Shift 연타**: 빠르게 여러 엘리먼트를 선택할 때 유용
2. **양손으로 누르기**: 왼손 Alt, 오른손 Shift (또는 그 반대)
3. **Input 필드에서도 안전**: Alt + Shift는 텍스트 입력을 방해하지 않습니다!
4. **자동 해제**: 액션을 선택하면 자동으로 잠금이 해제되어 다음 엘리먼트로 이동 가능
5. **컬러 구분**: 각 버튼의 색상으로 빠르게 원하는 액션 찾기
   - 파랑(Click), 초록(Type), 주황(Extract), 보라(Wait)
6. **Pause 버튼**: 복잡한 페이지에서 하이라이트가 방해될 때

## 문제 해결

### 툴바가 안 보여요
- 페이지를 새로고침해보세요
- Pause 버튼이 눌려있는지 확인하세요

### 버튼을 클릭할 수가 없어요
- **Alt + Shift (또는 Option + Shift)를 눌러서 잠금하세요!** 툴바가 주황색으로 변하면 클릭 가능합니다

### 사이드패널이 안 보여요
- Chrome 우측 상단 확장 프로그램 아이콘 클릭
- 또는 `chrome://extensions` → Automation Wizard → 세부정보 → 사이드 패널 권한 확인

### Run이 작동하지 않아요
- 콘솔을 열어서 에러를 확인하세요 (F12)
- 선택자가 여전히 유효한지 확인하세요
